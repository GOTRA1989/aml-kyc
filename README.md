# Automated RegTech AML/KYC Onboarding Platform (Web 1)

An end-to-end regulatory technology (RegTech) platform designed to automate customer onboarding and compliance auditing based on Tier-1 international banking standards.

## 🚀 Live Demo
🔗 **AML/KYC Hub:** https://aml-kyc.lovable.app/

## 📌 Key Features
- **Automated Onboarding:** Multi-layered workflows for Individual and Corporate entities with global jurisdictional screening (High-Risk, FATF Grey-listed, and Tax Haven dropdowns).
- **Analyst Review Dashboard:** Centralized compliance hub to audit Sanctions/PEP screening, analyze Adverse Media, and assign Customer Risk Ratings (CRR).
- **Core Banking Integration:** Features an automated "Create Bank Account" trigger that bridges verified profile data straight into the transaction monitoring system (Web 2).

## 🔄 Cross-Platform Data Workflow
1. Customer submits identity and corporate data on this platform (**Web 1**).
2. Compliance Analyst reviews the profile and clicks **"Approve & Create Bank Account"**.
3. **Web 1** securely packages the customer’s Name and Country, executing a deep-link redirection via URL parameters directly to the Core Banking Portal (**Web 2**).
